Jawaban Soal 1:

Web Server: Web server adalah perangkat keras atau perangkat lunak yang menyimpan, mengelola, dan mengirimkan halaman web kepada pengguna yang mengaksesnya melalui internet. Perangkat lunak server web umumnya mencakup Apache, Nginx, Microsoft IIS, dan lainnya. Domain: Domain adalah alamat unik yang digunakan untuk mengidentifikasi sebuah situs web di internet. Misalnya, dalam "www.example.com," "example.com" adalah domain. Domain menggunakan sistem Domain Name System (DNS) untuk menerjemahkan nama domain menjadi alamat IP yang dapat dimengerti oleh komputer. Hosting: Hosting merujuk pada tempat penyimpanan dan pengelolaan file-file website. Layanan hosting menyediakan server dan sumber daya yang diperlukan agar website dapat diakses secara online. Ada beberapa jenis hosting, termasuk shared hosting (berbagi sumber daya server dengan situs web lain), VPS hosting (Virtual Private Server), dan dedicated hosting (mendapatkan server sendiri). Hubungan Antara Web Server, Domain, dan Hosting: Domain dan Web Server: Saat seseorang memasukkan alamat domain ke dalam browser, permintaan itu dikirim ke server DNS. Server DNS akan mengembalikan alamat IP terkait dengan domain tersebut. Alamat IP ini digunakan oleh browser untuk mengarahkan permintaan ke server web yang sesuai. Domain dan Hosting: Hosting menyimpan semua file website, termasuk halaman HTML, gambar, dan skrip. Ketika seseorang mengakses domain, server hosting akan memberikan file-file tersebut kepada browser pengguna. Hubungan antara domain dan hosting dijelaskan oleh konfigurasi DNS, di mana DNS records menentukan alamat IP server hosting yang terkait dengan domain. Web Server dan Hosting: Web server adalah perangkat lunak atau perangkat keras yang mengelola permintaan dan respon HTTP. Server hosting menjalankan perangkat lunak server web ini untuk menyajikan konten website kepada pengguna.
